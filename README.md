## Описание программы
GitHub User Finder - это графическое приложение на Python для поиска пользователей GitHub с использованием официального GitHub API. Программа позволяет:

- Искать пользователей GitHub по имени
- Просматривать информацию о пользователе (аватар, имя, компания, локация, репозитории, подписчики)
- Добавлять пользователей в избранное
- Сохранять и загружать список избранных пользователей в JSON
- Открывать профиль пользователя в браузере
- Работать с асинхронными запросами (без зависания интерфейса)
